# Home

This directory contains the primary landing page interface for EpiRadar.

## Key Files
- `Home.tsx`: The main public landing page and marketing entry point.

## Functionality
- **Hero Section**: Introduces the platform mission ("Predict outbreak hotspots...") alongside an animated 3D `Orb` component. Features immediate calls to action to "Sign in" or view the "Public Portal".
- **About Platform**: High-level value propositions, detailing how EpiRadar aggregates hospital data and applies spatial clustering to predict health threats.
- **Workflow / How It Works**: A visual four-step breakdown of EpiRadar's operational loop: Data Ingestion -> Spatial Clustering -> Risk Scoring -> Alert & Action.
- **Purpose**: Defines the overarching goal of the project (Proactive intelligence for a healthier world) including stat grids for early detection, resource efficiency, and public safety.
