# Hospital

This directory is strictly allocated for the healthcare provider and administrative view of the EpiRadar system.

## Key Files
- `HospitalDashboard.tsx`: The primary dashboard interface for logged-in hospital personnel.

## Functionality (WIP / Scope)
While the public portal gives broad geographical alerts, the Hospital Dashboard is designed to be internal-facing for medical facilities. Its purpose is to:
- Review incoming predictive analytics concerning local outbreak surges.
- Monitor critical resource forecasts (e.g., forecasting supply needs up to 72 hours in advance).
- Coordinate directly with regional alerts triggered by the central EpiRadar spatial clustering engine.
